# 回歸情境 Demo 網頁規劃

## 目標
建立一個教學型 Demo 網頁，快速說明常見回歸模型適用的商業/生活情境，並對應到你目前專案中的範例程式。

## 受眾
- 初學機器學習的學生
- 想快速選模型的資料分析初學者

## 頁面規劃
1. 首頁 Hero
- 一句話說明：用情境選回歸，不再盲選模型
- CTA：開始探索模型

2. 模型情境卡片區
- 線性回歸：廣告預算與銷售額預測
- 多項式回歸：非線性關係（速度與煞車距離）
- Ridge 回歸：特徵多且共線性高時更穩定
- Lasso 回歸：需要特徵篩選時
- SVR：小樣本、非線性且需要容忍誤差
- 邏輯回歸（分類）：是否購買、是否流失

3. 選型小指南
- 目標是連續值 -> 回歸
- 特徵很多且容易過擬合 -> Ridge/Lasso
- 關係彎曲明顯 -> 多項式或 SVR

4. 範例程式對照區
- 顯示目前資料夾內對應的 Python 檔名（可後續再加連結）

## 後續可擴充
- 加入互動式滑桿（樣本噪音、正則化強度）
- 加入圖表展示（散點圖、回歸曲線）
- 串接後端 API 即時訓練與預測

## 目前已完成
- 卡片按鈕可呼叫本機 API，執行對應 Python 範例程式
- 頁面可顯示 STDOUT、STDERR 與退出碼

## 啟動方式
在 RegressionDemo 根目錄執行：

```powershell
python regression-scenarios-demo-web/server.py
```

打開：

```text
http://127.0.0.1:5500
```

備註：
- API 端點為 `/api/run?model=...`
- 程式逾時預設為 30 秒
- 已設定 `MPLBACKEND=Agg`，避免繪圖視窗阻塞執行
