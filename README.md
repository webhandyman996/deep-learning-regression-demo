# 回歸方法教學範例（機器學習基礎）

🔗 **教學網頁：https://webhandyman996.github.io/deep-learning-regression-demo/**

本 repo 為機器學習基礎單元的教學範例程式，透過經典回歸演算法，讓學生建立對監督式學習、模型訓練與評估的基本概念。

## 內容

| 主題 | 說明 |
| --- | --- |
| 線性回歸 | 最基礎的回歸模型，示範資料擬合與預測 |
| 多項式回歸 | 以煞車距離為例，示範非線性關係的擬合 |
| 嶺回歸（Ridge Regression） | 以廣告與銷售額關係為例，示範 L2 正則化如何抑制過擬合 |
| 套索回歸（Lasso Regression） | 以房屋價格預測為例，示範 L1 正則化與特徵篩選 |
| 支持向量回歸（SVR） | 以汽車油耗預測為例，示範核函數與參數調整（含互動式滑桿版本） |
| 邏輯回歸 | 以顧客購買行為預測為例，示範分類問題的回歸應用 |

部分主題附有進階/互動版本（檔名標示「2」），讓學生能實際調整參數觀察模型變化。

## 環境需求

Python 3.x，需安裝 `scikit-learn`、`matplotlib`、`numpy` 等套件。
